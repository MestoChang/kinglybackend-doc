# 會員資料(初步完成)
> - 檢視會員個人/帳號資料
> - 檢視交易/留言/瀏覽/登入/違規操作紀錄
> - 封鎖/解除封鎖會員
> - 屏蔽會員違反善良風俗的照片

!> 目前已完成會員註冊功能（僅紀錄帳號密碼），其他會員資料相關功能尚未開始製作；前端相關功能為二階段。

## 頁面元件

| 項目 | 類型 | 操作 | 系統回應與處理邏輯 |
| --- | --- | --- | --- |
| 封鎖會員 | Button | Click | 停止會員權限 |
| 解除封鎖 | Button | Click | - |
| 屏蔽照片 | Button | Click | 停止後顯示系統預設照片 |
| 交易紀錄 | Table | - | 點選後跳轉至該筆交易紀錄 |
| 留言紀錄 | Table | - | 點選後跳轉至該筆留言紀錄 |
| 瀏覽紀錄 | Table | - | - |
| 登入紀錄 | Table | - | - |
| 違規操作紀錄 | Table | - | - |

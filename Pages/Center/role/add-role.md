# 新增身份
> 設定身份的功能權限，以方便後續設定帳號的功能權限。

![Alt text](asset/add-role.png)


## 頁面元件
| 項目 | 類型 | 操作 | 系統回應與處理邏輯 |
| --- | --- | --- | --- |
| 名稱 | Input | Type | - |
| 權限級別 | Select | Select | 設定後不可更改 |
| 管理平台 | Select | Select | - |
| 英文名稱 | Input | Type | - |

!> 身份的權限級別會影響帳號是否可設定這個身份，若帳號的權限級別為 B 級，無法設定 A 級的身份。
